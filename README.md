## 🏆 Projetos em Destaque

### 🔬 OdontoForense
Sistema web completo para gerenciamento de casos periciais odontológicos com:
- Cadastro de vítimas e casos
- Geração de laudos com IA
- Exportação em PDF
- Dashboard administrativo

**Tecnologias:** Node.js, MongoDB, Express, HTML5, CSS3, JavaScript

[![Ver Projeto](https://img.shields.io/badge/🔗_Repositório-4682B4?style=for-the-badge)](https://github.com/seu-usuario/odontoforense)

### 🧠 Laudo IA Generator
Ferramenta inovadora que utiliza OpenAI para:
- Auxílio na criação de laudos técnicos
- Sugestões automáticas baseadas em casos anteriores
- Templates personalizáveis

**Status:** Em desenvolvimento ativo
